# INCUBALOG — Monitor Inteligente de Incubação

Sistema de monitoramento em tempo real para incubadoras avícolas da Unidade 1020.

## Acesso
Acesse o aplicativo em: **https://SEU_USUARIO.github.io/incubalog/**

## Funcionalidades
- Extração automática de dados por foto da tela da incubadora (IA)
- Alertas em 3 níveis: T. Embrião, T. Ar, CO₂, Umidade, Ventilação, Viragem
- Leitura por QR Code de cada incubadora
- Exportação Excel e CSV do ciclo completo
- Banco de dados local com histórico de 18 dias

## Configuração
1. Acesse o app pelo link acima
2. Informe sua API Key da Anthropic (console.anthropic.com)
3. Escaneie o QR Code da incubadora ou selecione manualmente
4. Inicie o ciclo e comece as leituras

- Sala 2: Inc. 19–36
- Sala 3: Inc. 37–54
